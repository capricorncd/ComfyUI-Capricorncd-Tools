# Audio Timeline（音频时间轴）

**分类：** `Capricorncd`

一个功能完整的音频波形编辑器，集成图像关键帧素材轨道、每片段提示词和结构化 JSON 输出。专为音频驱动的视频生成流程设计——音频的每个区间对应一个生成的视频片段。

## 面板介绍

### 波形面板

- 基于 [WaveSurfer.js](https://wavesurfer.xyz/) 的可视化波形
- 拖动左右两侧的黄色手柄来定义修剪区间（`start_time` / `end_time`）
- 点击波形任意位置移动时刻线
- `▶` 按钮播放 / 暂停已修剪的音频片段
- `←` / `→` 方向键以一帧为步长微调当前选中的修剪手柄

### 素材轨道

每个素材代表一个生成区间：包含时间范围、起始关键帧图片、可选的结束关键帧图片以及该片段的提示词。

轨道分为两条：

| 轨道 | `track` 值 | 排列方式 | 行为 |
|------|-----------|----------|------|
| 主轨道 | `0` | 始终紧密排列 | 移动、调整大小或删除后自动从左对齐，不留间隙 |
| 副轨道（叠加轨） | `1` | 自由定位，允许间隙 | 素材保持放置的位置不变；与其他副轨道素材重叠会被拒绝 |

副轨道素材渲染在上层（`z_index 2`），会**遮挡**其覆盖范围内的主轨道内容（`z_index 1`）——导出时，被遮挡的主轨道区间会从 `data_json` / `total_frame_count` 中切除，因此即使最终只生成一段视频，时间轴也表现为双层合成的效果。

**添加素材**

| 方式 | 行为 |
|------|------|
| `＋ Add Image` 按钮 | 打开图片选择器；若时刻线处主轨道空闲则插入主轨道，否则插入副轨道，若两条轨道都被占用则提示无法插入 |
| 双击主轨道 | 在该时间位置添加一个主轨道空素材（未分配图片） |
| 双击副轨道 | 在该时间位置添加一个副轨道素材；若该位置已有副轨道素材则提示无法插入 |

主轨道素材始终紧密排列——移动、调整大小或删除后自动从左对齐，不留间隙。副轨道素材不受此约束。

**素材缩略图与徽标**

- 缩略图显示起始关键帧图片（如已分配）
- `[首]` 徽标：仅设置了首帧；鼠标悬停可预览
- `[首尾]` 徽标：首帧和尾帧均已设置；鼠标悬停可同时预览两张图

**右键上下文菜单**

| 选项 | 快捷键 | 说明 |
|------|--------|------|
**多选（Ctrl+点击）**

按住 `Ctrl` 点击素材可逐个加入/移出选中集合。多选后右键任意选中的素材，显示多选专属菜单：

| 选项 | 快捷键 | 条件 | 说明 |
|------|--------|------|------|
| 合并 | — | 所选素材连续相邻 | 保留首个素材，将其结束时间延长至最后一个素材的结束时间，删除其余素材 |
| 禁用/启用选中项 | `Ctrl+B` | 任意 | 若所有选中项均已禁用则全部启用，否则全部禁用 |

非相邻素材不显示"合并"选项。

**单选右键菜单**

| 选项 | 快捷键 | 说明 |
|------|--------|------|
| 替换素材 | — | 打开图片选择器，替换首帧图片 |
| 选择尾帧图片 | — | 打开图片选择器，设置尾帧图片 |
| 首尾帧交换 | — | 交换首帧与尾帧图片（仅在两者均已设置时显示） |
| 分割素材 | — | 从当前时刻线位置将素材拆分为两个独立素材，首帧归左段、尾帧归右段（仅在时刻线位于素材内部时显示） |
| 移到主轨道 / 移到副轨道 | — | 将素材移动到另一条轨道；若副轨道目标位置已被占用则拒绝移动 |
| Disable / Enable | `Ctrl+B` | 切换素材的禁用状态 |
| Disable Others / Enable Others | `Ctrl+G` | 禁用其他所有素材；若其他素材已全部禁用则重新启用 |
| 复制 | — | 将该素材复制到内部剪贴板 |
| 删除 | `Delete` | 删除该素材 |
| 清除尾帧图片 | — | 移除尾帧图片（仅在已设置尾帧时显示） |

### 片段禁用 / 启用

禁用功能让你可以**只重新生成某一段，而不影响时间轴其他部分**。

典型流程：
1. 全片段运行一次完整生成
2. 某一段效果不满意，需要重做
3. 选中该片段 → `Ctrl+G` 禁用其他所有片段（只保留目标片段启用）
4. 重新运行生成——只处理当前启用的片段
5. 再次对同一片段按 `Ctrl+G`，恢复其他片段的启用状态

禁用的片段：
- 以淡化 + 文字删除线样式显示
- **不会**出现在 `data_json` 输出中，也不计入 `total_frame_count`
- 不影响启用片段的时间，`clips_length` 只统计活跃片段

### 提示词区域

- 选中素材后，提示词区域切换为**每片段**模式（标签变为金色）
- 未选中任何素材时，文本框编辑**全局提示词**
- 每片段提示词优先；未设置每片段提示词的素材将使用全局提示词

## 资源目录（`assets_dir`）

`assets_dir` 是音频文件与关键帧图片的共用资源目录。

- 图片选择器会列出该目录下的所有图片文件
- 导入时，导出 JSON 中保存的音频文件名将相对此目录解析
- 输入框旁边会显示一个 `！` 提示图标，悬停可查看说明

**↻ 刷新** 可在不关闭选择器的情况下重新扫描目录。

## 导入 / 导出

| 按钮 | 行为 |
|------|------|
| **Export** | 将完整时间轴配置（所有参数值含音频文件名 + 素材列表）保存为 `{音频名}_{yyyyMMdd_HHmmss}.json` |
| **Import** | 从之前导出的 `.json` 文件中还原所有参数值（含音频）和素材列表，并自动触发波形重载 |

## 键盘快捷键

在波形或素材轨道任意位置点击以获得焦点。`Ctrl+B` 和 `Ctrl+G` 在选中素材时始终有效，无需关注焦点。

| 按键 | 操作 |
|------|------|
| `Space` | 播放 / 暂停时间轴 |
| `←` / `→` | 移动时刻线一帧；或微调选中的修剪手柄 |
| `Q` | 将选中素材的左边缘修剪到时刻线位置 |
| `W` | 将选中素材的右边缘修剪到时刻线位置 |
| `Delete` / `Backspace` | 删除选中素材 |
| `Ctrl+C` | 复制选中素材 |
| `Ctrl+V` | 将剪贴板素材粘贴到其原轨道（主轨道或副轨道）的末尾 |
| `Ctrl+B` | 禁用 / 启用选中素材 |
| `Ctrl+G` | 禁用 / 启用其他所有素材 |
| `Escape` | 取消选中 / 关闭选择器或上下文菜单 |

## 输入参数

| 名称 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `audio` | FILE | — | ComfyUI 输入目录中的音频或视频文件 |
| `fps` | FLOAT | 24.0 | 帧率 |
| `width` | INT | 720 | 输出视频宽度（写入 `data_json`） |
| `height` | INT | 1280 | 输出视频高度（写入 `data_json`） |
| `assets_dir` | STRING | — | 存放关键帧图片和音频文件的目录，图片选择器和导入时使用 |
| `one_shot` | BOOLEAN | true | 一镜到底模式：非最后一个素材的尾帧自动设为下一个素材的首帧 |
| `global_prompt` | STRING | — | 无每片段提示词的素材使用的默认提示词 |
| `trim_offset` | INT | 1 | 附加到 `AUDIO` 输出末尾的秒数，用于淡出/叠化；**不影响** `data_json` 时间或帧数 |

## 输出参数

| 名称 | 类型 | 说明 |
|------|------|------|
| `trimmed_audio` | AUDIO | 修剪后的音频，末尾延伸 `trim_offset` 秒 |
| `fps` | FLOAT | 帧率 |
| `one_shot` | BOOLEAN | 一镜到底标志 |
| `width` | INT | 视频宽度 |
| `height` | INT | 视频高度 |
| `global_prompt` | STRING | 全局提示词字符串 |
| `data_json` | STRING | 完整配置 JSON（仅包含启用的片段） |
| `clips_length` | INT | 启用的片段数量 |
| `total_frame_count` | INT | 所有启用片段的总帧数 |
| `clips_audio` | AUDIO | 仅由启用片段拼接而成的音频（排除禁用片段和间隙） |
| `frame_seq_dir` | STRING | 序列帧临时目录（`output/temp/capricorncd-frame-sequences`），首次运行时创建，每次运行前完全清空 |

## `data_json` 数据结构

```json
{
  "audio_path": "/absolute/path/to/audio.mp3",
  "trim_start_ms": 0,
  "trim_end_ms": 30000,
  "total_frame_count": 720,
  "fps": 24.0,
  "width": 720,
  "height": 1280,
  "one_shot": true,
  "global_prompt": "cinematic, 4k",
  "clips": [
    {
      "start_ms": 0,
      "end_ms": 5000,
      "start_image": "/absolute/path/to/frame_001.jpg",
      "end_image": "/absolute/path/to/frame_010.jpg",
      "prompt": "close up portrait",
      "use_global_prompt": false,
      "z_index": 1
    }
  ]
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `start_ms` | number | 片段开始时间（毫秒），相对于修剪起点 |
| `end_ms` | number | 片段结束时间（毫秒），相对于修剪起点 |
| `start_image` | string | 首帧图片的绝对路径（未设置时为空字符串） |
| `end_image` | string | 尾帧图片的绝对路径；一镜到底模式下，非最后片段自动使用下一片段的 `start_image` |
| `prompt` | string | 每片段提示词（未设置时为空字符串） |
| `use_global_prompt` | boolean | 为 `true` 时该片段使用全局提示词（显式设置或每片段提示词为空时均为 `true`） |
| `z_index` | number | 主轨道素材为 `1`，副轨道素材为 `2` |

禁用的片段**不会**写入 `clips`，`total_frame_count` 仅统计启用片段的帧数之和。主轨道中被启用的副轨道素材遮挡的区间会被切除（若完全被遮挡则整体丢弃），因此一个主轨道 UI 素材在 `clips` 中可能对应零个、一个或两个条目。
