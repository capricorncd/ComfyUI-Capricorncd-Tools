# Prompt Input / Rich Prompt Input（提示词输入）

**分类：** `Capricorncd`

两个支持行注释的提示词编辑节点。两者都会从输出中剔除以 `#` 开头的行，让你可以在文本框中直接保留笔记、已禁用的变体或备选提示词，而不会影响生成结果。

---

## Prompt Input（提示词输入）

简洁的多行文本框，支持注释切换。

**功能**
- 以 `#` 开头的行被视为注释，不写入输出
- `Ctrl+/` 为当前行或所有选中行切换 `#` 注释标记

| 输出 | 类型 | 说明 |
|------|------|------|
| `prompt` | STRING | 所有非注释行以换行符拼接后的文本 |

---

## Rich Prompt Input（富文本提示词输入）

通过透明覆盖层实时渲染语法高亮的增强版编辑器。

**功能**
- 注释行在编辑器中以较暗的颜色显示
- `Ctrl+/` 为当前行或选中内容切换 `#`
- 粘贴时自动剥离富文本格式（仅保留纯文本）
- 注释标记不写入输出，只有正文内容向下传递
- 可选在输出开头 / 结尾追加空行
- **历史记录**、**预设** 合并为按钮「预设/历史记录」，模态框内 Tab 切换（保存在浏览器 `localStorage`）

### 历史记录 / 预设

| 操作 | 行为 |
|------|------|
| 预设 / 历史记录 Tab | 同一模态框内切换两类列表 |
| 历史记录 | 最近提示词（节点每次成功执行后自动保存；也可手动保存） |
| 预设 | 手动保存的命名提示词片段 |
| 插入 | 插入到最后记录的光标位置；若无光标记录则追加到末尾 |
| 替换 | 替换输入框全部内容 |
| 导出 / 导入 | 针对当前 Tab 的 JSON 往返；导入时可合并或替换 |

| 输出 | 类型 | 说明 |
|------|------|------|
| `prompt` | STRING | 移除 `#` 注释标记后的有效行文本 |
