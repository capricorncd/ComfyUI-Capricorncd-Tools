# 图像批次计数 / 按索引取图（Image Batch Count / Image From Batch Index）

**分类：** `Capricorncd`

用于处理 `IMAGE` 批次的辅助节点。

---

## 图像批次计数（Image Batch Count）

返回批次中的图片数量。

| 输入 | 类型 | 说明 |
|------|------|------|
| `images` | IMAGE | 输入批次 |

| 输出 | 类型 | 说明 |
|------|------|------|
| `count` | INT | `images.shape[0]` |

---

## 按索引取图（Image From Batch Index）

按索引从批次中取单张图片。

| 输入 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `images` | IMAGE | — | 输入批次 |
| `index` | INT | `0` | 批次索引。负值从末尾计数（`-1` = 最后一帧） |

| 输出 | 类型 | 说明 |
|------|------|------|
| `image` | IMAGE | 单张图像批次（`shape[0] == 1`） |

---

## 说明

- 正索引越界时会被钳制到最后一张。
- 负索引会先换算再钳制，因此 `-1` 始终表示最后一帧。

---

## 示例

```
IMAGE 批次（48 帧）
  ├── 图像批次计数    → count = 48
  └── 按索引取图
        index = -1   → 仅最后一帧
```
